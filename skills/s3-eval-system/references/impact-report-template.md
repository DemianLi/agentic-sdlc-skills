# Impact Report Template

File path: `docs/arch/YYYY-MM-DD-<topic>-impact.md`

```markdown
## Impact Report — <Iteration Topic>

### Breaking Changes (🔴)
- <component>: <what changes> → <migration needed>

### Additive Changes (🟡)
- <component>: <what is added>

### Technical Debt to Resolve First
- <file/area>: <debt description>

### Recommended Approach
<1-2 sentences summarising the safest implementation path>
```

Commit command: `git add docs/arch/ && git commit -m "arch: add impact report for <topic>"`
