## Usage

### Python API

```python
from slugify import slugify

# Basic usage
slugify("Hello World")  # → "hello-world"

# Custom separator
slugify("Hello & World", separator="_")  # → "hello__world"
```

### Command Line

```bash
# Default hyphen separator
python -m slugify "Hello World"  # → hello-world

# Custom separator
python -m slugify "Hello World" --separator _  # → hello_world
```
