## Task Breakdown

### TASK-1: Project scaffold
- Create `src/mdtoc/__init__.py`, `src/mdtoc/core.py`, `src/mdtoc/__main__.py`
- Create `tests/test_core.py`, `tests/test_cli.py` (empty stubs)
- Create `pyproject.toml` with `[project]` metadata and `[project.scripts]`
- **Acceptance**: `python -m mdtoc` runs without ImportError

### TASK-2: Implement `parse_headers()` (REQ-1)
- ATX heading regex: `re.match(r'^(#{1,6}) (.+)', line)`
- Code fence tracking (`in_code_block` flag)
- Strip heading text
- **Acceptance**: AC-1.1 through AC-1.4 pass

### TASK-3: Implement `generate_toc()` (REQ-2)
- Filter by `max_level`
- Indent: `"  " * (level - 1)`
- Slug: lowercase, spaces → `-`, strip `[^a-z0-9-]`
- **Acceptance**: AC-2.1 through AC-2.4 pass

### TASK-4: Implement `insert_toc()` (REQ-3)
- Marker detection outside code blocks
- Replace-between-markers path
- Prepend-if-absent path
- **Acceptance**: AC-3.1 through AC-3.4 pass

### TASK-5: Implement CLI `__main__.py` (REQ-4)
- `argparse` subcommand `generate`
- `--max-level`, `--in-place` flags
- File I/O + error handling (exit 1 on FileNotFoundError)
- **Acceptance**: AC-4.1 through AC-4.5 pass

---

## Dependency DAG

```
TASK-1 (scaffold)
   └─ TASK-2 (parse_headers)
   └─ TASK-3 (generate_toc)
   └─ TASK-4 (insert_toc)
         └─ TASK-5 (CLI — depends on TASK-2,3,4)
```

TASK-2, TASK-3, TASK-4 can be implemented in parallel after TASK-1.
TASK-5 depends on TASK-2 + TASK-3 + TASK-4.
